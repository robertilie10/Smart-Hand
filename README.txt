Smart Hand

Smart Hand is an innovative application that uses hand detection technology to control the brightness and volume of your laptop or computer directly from hand movements. This application is designed to offer an intuitive and easy-to-use interface for quickly adjusting these settings without the need to touch the keyboard or mouse.

Features

- Brightness Control: Adjust the screen brightness by varying the distance between your thumb and index finger.
- Volume Control: Control the system volume using the same gestures.
- Intuitive Graphic Interface: Select brightness and volume options with a simple hand gesture.
- Additional Options: Access an information screen or close the application through virtual buttons.

System Requirements

- Operating System: Windows 10 or later.
- Hardware: Functional webcam.

Installation

1. Extract Files: Unzip the files.
2. Execution: Double-click the `Smart Hand.exe` file to launch the application.

Usage

1. Launch: After opening the application, the webcam will activate and start detecting your hand.
2. Brightness and Volume Control:
   - Select Function: Hold your index finger over the brightness or volume icon for 3 seconds to select them.
   - Adjustment: Increase or decrease the value by moving your thumb away from or closer to your index finger.
   - Interrupt: Hold your index finger over the STOP icon to deselect the brightness or volume function.
3. Info Button: Select the `INFO` icon to display additional information about using the application.
4. Close: Use the `ESC` button on the interface or press the `ESC` key to close the application.

Troubleshooting

- Hand Detection Issues: Ensure that your hand is fully visible to the camera and that the lighting in the room is adequate.
- Launch Error: Check that all necessary files are in the same folder as the executable.
